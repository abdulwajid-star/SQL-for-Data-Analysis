# SQL for Data Analysis – E-commerce Dataset

## Project Overview

This project demonstrates the use of SQL to extract, manipulate, and analyze data from an e-commerce dataset using MySQL. The objective was to practice essential SQL concepts used in data analysis, including filtering, sorting, grouping, aggregation, subqueries, views, and indexing.

---

## Objectives

- Extract data using SQL queries
- Analyze customer and product information
- Perform aggregations and summarizations
- Create reusable SQL views
- Optimize query performance using indexes

##  Sample Queries

### Display all records

```sql
SELECT * FROM ecommerce_dataset;
```

### Filter customers by country

```sql
SELECT *
FROM ecommerce_dataset
WHERE country='India';
```

### Sort products by price

```sql
SELECT product_name, unit_price
FROM ecommerce_dataset
ORDER BY unit_price DESC;
```

### Revenue by category

```sql
SELECT
category,
SUM(quantity * unit_price) AS Revenue
FROM ecommerce_dataset
GROUP BY category;
```

### Customers with above-average ratings

```sql
SELECT first_name, rating
FROM ecommerce_dataset
WHERE rating >
(
SELECT AVG(rating)
FROM ecommerce_dataset
);
```
